## Описание проекта

**Генератор случайных цитат** — это удобное desktop-приложение с графическим интерфейсом, созданное на Python с использованием библиотеки Tkinter. Программа позволяет получать вдохновляющие и мотивирующие цитаты по нажатию одной кнопки.

### Основные возможности

- ✅ **Генерация случайных цитат** — одна кнопка и новая цитата готова
- ✅ **Фильтрация по автору** — выбирайте цитаты только любимых авторов
- ✅ **Фильтрация по теме** — ищите цитаты на нужную тему (мотивация, успех, счастье и т.д.)
- ✅ **История цитат** — все сгенерированные цитаты сохраняются с датой и временем
- ✅ **Добавление своих цитат** — пополняйте базу собственными любимыми цитатами
- ✅ **Проверка ввода** — защита от добавления пустых строк
- ✅ **Сохранение данных** — вся информация сохраняется в JSON файлы

### Как это работает

1. Запускаете программу
2. Нажимаете «Сгенерировать цитату»
3. Получаете случайную цитату из базы
4. Цитата автоматически сохраняется в историю
5. При желании можете добавить свою цитату
