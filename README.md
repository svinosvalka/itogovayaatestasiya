## Описание программы
Random Quote Generator — это десктопное приложение, которое хранит коллекцию цитат с авторами и темами, выдает случайную цитату по нажатию кнопки, запоминает историю показанных цитат, позволяет фильтровать по автору или теме, добавлять новые цитаты и сохраняет историю между запусками в JSON-файл.
## Требования для запуска
- Python 3.10 или выше
- Библиотеки 'pip install matplotlib requests'
